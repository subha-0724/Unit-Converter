# Unit-Converter

This is a simple unit conversion website created using HTML, CSS, and JavaScript. It allows users to easily convert values between different units such as length, weight, and temperature. The interface is user-friendly and designed with basic styling to make it easy to use. JavaScript is used to perform the conversion calculations dynamically based on user input.

This project was developed as my first hands-on practice after learning the basics of HTML, CSS, and JavaScript. It helped me understand how to connect structure, design, and functionality in a real-world application.
